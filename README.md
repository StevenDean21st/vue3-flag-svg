# Vue 3 Flag SVG

SVG Flag icons based one Vue 3.x .

## Generator Icons

This package is based on [flag-icons](https://www.npmjs.com/package/flag-icons).

To generate all flag icons, run:

```bash
npm install

npm run generate
# or
node generator/generator.js
```

To preview all icons, run:

```bash
npm run test
# or
vite test
```
